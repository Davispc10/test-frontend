## ComicCard Component

### Props:

```ts
type ComicCardProps = {
  title: string;
  description: string;
  thumbnail: string;
};
```

### Example:

```tsx
<ComicCard
  title: "Maestro: World War M (2022)";
  description: "CHAOS WAR TIE-IN! When the King of Chaos";
  thumbnail: "http://i.annihil.us/u/prod/marvel/i/mg/d/90/62d0454381496.jpg";
/>
```
